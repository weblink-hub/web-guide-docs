# web-dev-notes
## 经典唐诗完整赏析文库
唐诗是中华古典文学瑰宝，流传千年依旧深受大众喜爱。本文库整理唐代知名诗人经典诗作，覆盖中小学必背篇目、课外拓展名篇，包含原文、白话译文、字词注释、创作背景、诗句赏析五大模块，一站式满足诗词学习需求。

### 一、收录诗人范围
文库收录李白、杜甫、王维、孟浩然、白居易、刘禹锡、王之涣、王昌龄等盛唐、中唐代表诗人作品，兼顾山水田园诗、边塞诗、送别诗、咏物诗四大主流题材，分类清晰便于查找。

### 二、内容排版说明
1. 原文分段排版，生僻字标注读音，通假字单独注解；
2. 译文直白通俗，无晦涩古文，适合学生、文学爱好者自学；
3. 赏析部分拆解意象、修辞、作者心境，辅助理解诗词深层情感；
4. 补充创作年代、作者生平小故事，丰富阅读趣味性。

### 三、适合阅读人群
1. 中小学生，用于语文背诵、古诗文阅读理解复习；
2. 文学爱好者，日常摘抄、积累文案素材；
3. 语文教师，备课查找诗词完整讲解资料。

### 持续更新计划
文库每周更新5-8首冷门优质唐诗，完善配套考题、默写重点标注，长期更新古典文学相关资料，喜欢传统文学可收藏页面，随时查阅拓展学习资源。
